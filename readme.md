# ✨ Descripción

AGATA & JARVIS IA es un asistente virtual desarrollado en Python utilizando la API de Gemini Live.

El proyecto combina:
- 🎙️ Conversaciones por voz en tiempo real
- 👁️ Visión artificial
- 🌐 Automatización web
- 💻 Generación de código
- 🖥️ Control del sistema operativo
- 📂 Gestión de archivos
- 🤖 Automatización inteligente de tareas

Cuenta con dos asistentes inteligentes:

| IA | Función |
|---|---|
| 🤖 **Jarvis** | Programación, automatización y control del sistema |
| 🎨 **Agata** | Diseño, documentos y presentaciones |

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/Angel-ETECH/AGATA-JARVIS-V2.git
cd AGATA-JARVIS-V2
````

---

## 2️⃣ Instalar dependencias

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Instalar Playwright

```bash
python -m playwright install
```

---

## 4️⃣ Ejecutar proyecto

```bash
python main.py
```
